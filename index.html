Bienvenue sur le Scraper EUR-Lex

Cette API vous permet de récupérer des données de recherche spécifiques sur le site EUR-Lex.

Utilisation de l'API

Pour lancer le scraping de toutes les pages de résultats et afficher les données dans un tableau structuré, cliquez sur le bouton ci-dessous :

Endpoint pour la recherche complète :

GET /scrape-all-eurlex

Afficher les résultats structurés

Les données brutes seront également disponibles au format JSON en accédant directement à l'endpoint : /scrape-all-eurlex.
